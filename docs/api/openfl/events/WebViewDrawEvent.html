class WebViewDrawEvent

package openfl.events

extends Event

@:directlyUsed

Event dispatched when StageWebView completes a viewport capture request.

Depending on the target requested by drawViewPortAsync(), the event will populate bitmapData, pngImage, or jpegImage.
